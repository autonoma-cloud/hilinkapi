Home
=====

This documentation will provide you a guide to integrate HiLink API for Python 3.
This is a Python 3 module which can use to manipulate data/internet connection related operations
for *Huawei HiLink* modems which are having WebUI versions,

* 17.x.x.x
* 21.x.x.x
* 10.x.x.x

and I have tested successfully with,

* E8372h-320
* E8372h-155
* E3372h-320
* E3372h-153

modems.
